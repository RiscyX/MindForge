# Setup Guide

This document describes how to set up the project on **Windows** and **Linux** environments, using **XAMPP** or **LAMPP**

The application is a PHP + MySQL web application and requires a standard web stack with Composer support.

---

## 1. Prerequisites

### 1.1 Common requirements

Make sure you have the following installed:

-- **XAMPP with PHP 8.2.12**

Optional but recommended:

- **Mailtrap** account (for testing registration and password reset emails)
